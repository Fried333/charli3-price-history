Charli3 Price Oracle

Real-time and historical price data from Charli3's on-chain oracle feeds on Cardano

What is this?

Charli3 operates a pull oracle on Cardano — prices are published on-chain only when someone requests an update. This means there's no continuous price feed and no historical price API.

This tool solves that. It crawls the blockchain, finds every price datum Charli3 has ever published, decodes the Plutus CBOR data, and serves it via a simple REST API.

How it works:

  1. Scans the Charli3 oracle smart contract address on Cardano
  2. Finds all UTxOs carrying C3AS (Asset State) tokens with inline datums
  3. Decodes CBOR: Tag(121, [Tag(123, [{0: price, 1: valid_from, 2: valid_to}])])
  4. Indexes prices in a database with timestamps
  5. Serves via REST — one HTTP call instead of chain queries + CBOR parsing

Without Charli3's oracle, this tool has no data. Every price shown here was published by Charli3's oracle nodes after aggregating from multiple sources and cryptographically signing the result.

API Documentation

Latest price for a feed:

curl https://86-107-168-44.sslip.io/api/price/ADA-USD

Historical prices (up to 1000):

curl https://86-107-168-44.sslip.io/api/history/ADA-USD?limit=50

List all feeds with latest prices:

curl https://86-107-168-44.sslip.io/api/feeds

Overall statistics:

curl https://86-107-168-44.sslip.io/api/stats
